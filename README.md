# klipperconfig
Klipper config

My klipper config. This was a lot of searching to set up, hopefully making it easier for the next guy.

Hardware:
- Delta kinematics
- Duet Wifi
- Smart Effector bed probe
- Mains powered bed heatear
- Flex3drive extruder
- E3d hotend with PT100 Thermocouple attached to the Duet Temperature daughterboard.
